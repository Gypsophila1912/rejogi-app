## 0️⃣ 前提定義

| 項目         | 内容                                                                           |
| ------------ | ------------------------------------------------------------------------------ |
| 対象ユーザー | 立花祭で会計業務を行うサークルメンバー（会計長・レジ係）                       |
| 提供価値     | 立花祭の会計処理を自動化し、集計時間を削減しつつリアルタイムで売上を把握できる |
| コア体験     | 簡単操作で会計が完了し、売上が即時反映される                                   |
| 成功指標     | 計算ミス件数0件、売上リアルタイム確認100%を達成すること                        |
